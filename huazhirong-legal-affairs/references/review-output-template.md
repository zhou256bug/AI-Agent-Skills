# 合同审核输出模板

所有模式统一使用以下结构。**结论先行**，不线性铺垫。

---

## A 模式 — 海外经销

```markdown
## 合同风险分析 — [客户名] / [国家] / [新签|续签]

**我方立场**：乙方（Vendor）
**结论**：（一句话最大风险）

### 🔴 必须改
（条款原文 → 问题 → Playbook 依据 → 建议措辞）

### 🟡 注意
...

### 🟢 可接受
...

### 续签差异（如有）
| 条款 | 旧版 | 新版 | 等级 |

### 缺失项检查
- [ ] ...

### 建议下一步
```

---

## E 模式 — 解约终止

```markdown
## 解约/终止分析 — [对方] / [合同类型] / [国家]

**结论**：（一句话：能否解约、最早生效日、预估成本）

### 关键条款摘录
| 条款 | 内容 | 影响 |

### 方案对比
| 方案 | 成本 | 退出日 | 风险 |

### 🔴 程序风险
### 🟡 谈判要点
### 建议下一步
```

## G 模式 — 合资 / 股东协议 / 增资

```markdown
## 股东/合资协议审核 — [交易方] / [子公司或JV] / [增资|借款|JV]

### 出资结构对比
| 方式 | 税务 | 治理 | 建议 |

### 🟡 注意
### 🟢 可接受

### 建议董事会/老板下一步
```

## 归档路径

| 模式 | 路径 |
|------|------|
| A / E | `output/合同/审核记录/YYYY-MM-DD_对方名_合同审核.md` |
| B | `output/合同/采购/审核记录/YYYY-MM-DD_供方名_合同审核.md` |
| C | `output/合同/人事/审核记录/YYYY-MM-DD_姓名或岗位_合同审核.md` |
| F / G | `output/合同/股权/审核记录/YYYY-MM-DD_交易方_协议审核.md` |
