# 华智融采购合同 Playbook（我方 = 甲方 / Buyer）

> **适用**：向供应商采购原材料、元器件、委外加工、物流、软件/SaaS、检测认证服务等  
> **立场**：华智融为**甲方（买方）** — 与海外经销合同（我方乙方）立场相反  
> **决策权**：采购/供应链起草 → 本 Agent 审核 → 老板/采购负责人拍板

## Tier 1 — 必须坚守（Deal Breakers / 🔴）

### 1.1 标的、规格与验收

- 物料型号、版本、质量标准（引用图纸/BOM/规格书编号）
- **验收程序**：到货检验期限（建议 ≥ 15 个工作日）、不合格品处理
- **所有权转移**：与 Incoterms 一致；未验收合格前风险由供方承担

**RED**：规格笼统「等同样品」、无验收期、签收即视为合格。

### 1.2 交付与 Incoterms

- 明确交货地点、批次、交期；逾期违约金（建议 ≥ 0.5%/日，设上限）
- 华智融常用：**FCA 工厂** / **FOB** / **DDP**

**RED**：交期无约束、供方单方延期无罚则、Incoterms 与付款/风险条款矛盾。

### 1.3 质量保证

- 硬件/结构件：质保期建议 ≥ **12–24 个月**（从验收合格起）
- 批次不良率上限、RMA 流程、备品备件
- **关键元器件**：须符合华智融 BOM 与认证要求（3C 零部件、RoHS 等）

**RED**：质保 < 6 个月、排除隐蔽缺陷、供方单方解释验收标准。

### 1.4 知识产权（采购定制件/软件时）

- 供方背景 IP 归供方；**为华智融定制开发的成果归华智融**
- 供方保证不侵权；侵权时供方 indemnity + 替换/赔偿
- 禁止供方将华智融定制设计用于第三方竞品

**RED**：定制开发 IP 归供方、无侵权保证。

### 1.5 价格与调价

- 固定价期间（建议 ≥ 12 个月）或公式调价（原材料指数 + 提前 30 日通知）
- 供方单方涨价权 → 🔴

### 1.6 付款（买方偏好）

**首选**：验收合格后付款；或分期（预付 ≤ 30% + 到货付余款）

**RED**：100% 预付无担保、无履约保函/质保金。

### 1.7 责任与赔偿（买方要强）

- **供方赔偿**：因供方违约、缺陷产品、侵权导致华智融损失的，供方承担
- 华智融对供方的责任上限：宜设 cap（如合同金额 100%），排除间接损失
- 供方须赔偿：召回、客户索赔、监管罚款（因供方原因）

### 1.8 终止

- 华智融 **termination for convenience**：提前 **30–60 日**通知即可终止
- 供方违约：华智融可立即终止 + 索赔
- 终止后：在制品、专用物料、模具归属（华智融付费模具归华智融）

**RED**：华智融无便利终止权、模具/专用料无偿归供方。

### 1.9 管辖与争议

- 首选：**中国法 + CIETAC / 华南国仲**（深圳）
- 境外供方：可接受 **香港法 + HKIAC** 或 **新加坡法 + SIAC**

### 1.10 合规

- 反腐败、制裁合规、出口管制
- 审计权：华智融可审计供方与合同相关的记录

## Tier 2 — 强烈建议（🟡）

| 条款 | 买方立场 |
|------|----------|
| 保密 | 双向；华智融技术/BOM 保密期 ≥ 3 年 |
| 不可抗力 | 通知义务 + 超过 90 日可终止 |
| 分包/转包 | 须华智融书面同意（关键物料） |
| 产能保证 | MOQ、安全库存、优先供货 |
| 保险 | 供方产品责任险 |
| 数据保护 | 采购 SaaS/云服务时须 DPA |

---

## Tier 3 — 可让步

- 争议先友好协商 30 日
- 供方标准格式合同的 minor 定义调整
- 华智融付款账期 30–45 日（优质战略供方）

## 常见采购类型快查

| 类型 | 额外关注 |
|------|----------|
| 元器件/芯片 | 真伪、产地、ECCN、长单 cancel 窗口 |
| OEM/ODM 整机 | 认证责任划分、不良率、返修中心 |
| 结构件/包材 | 模具权属、打样费用 |
| 物流/货代 | 货损责任、保险受益人 |
| 软件/SaaS | 许可范围、SLA、数据出境、退出迁移 |
