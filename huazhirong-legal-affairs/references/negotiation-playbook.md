# 华智融谈判 Playbook（海外经销/代理）

> **适用**：海外 POS 终端经销/独家代理协议  
> **我方角色**：**乙方（Vendor / Licensor）** — 始终从乙方利益出发  
> **决策权**：本 Playbook 定义标准立场；最终商业决策由老板作出  
> **加载顺序**：仅适用于**海外经销/代理（我方乙方）**。采购见 `procurement-playbook.md`；用工见 `labor-cn-playbook.md`

|------|--------|

## Tier 1 — 必须坚守（Deal Breakers）

### 1.1 本土保护法 + 登记条款（Law 173 等）

**标准立场**：
- 续签时若**新增**当地代理保护法引用或主管机构登记授权 → 默认 🔴
- **老板确认策略（2026-05-29）**：可保留 Law 173 等引用，但必须同时写入**明确、可量化的终止触发条件**（just cause）
- 必须加：**本地法/登记不影响本合同约定的管辖法与仲裁条款**

**最低可接受终止触发条件（至少包含）**：
1. 未达约定最低采购量（连续 N 个考核期）
2. 逾期付款超过 15 日未补足
3. 违反竞业/排他区域销售
4. 违反保密义务
5. 虚假陈述或重大违约
6. 破产/清算/丧失经销资质

**RED 触发**：对方拒绝任何终止触发条件，或要求登记后无退出路径 → 暂停签署，升级老板。

**参考 redline 要点**：见 `latam-distributor-law.md` § Law 173。

### 1.2 管辖法与争议解决

**首选（GREEN）**：中国法 + CIETAC；或香港法 + HKIAC

**可接受（YELLOW）**：新加坡法 + SIAC（战略客户、对方强烈坚持时）

**RED 触发**：对方国法律专属管辖且无仲裁；远程不便法院 + 放弃仲裁；条款使本地保护法法院优先于约定仲裁

### 1.3 知识产权

- 乙方保留全部 IP；甲方仅获**区域内推广销售产品之目的**的非独占许可
- SDK/API/品牌使用范围必须书面限定 territory + term
- 禁止：甲方获得衍生 IP、反向工程后归属甲方、宽泛 feedback 条款

**RED 触发**：IP 转让、无限许可、或授权范围覆盖全球研发/生产。

### 1.4 独家与最低采购量匹配

- 独家区域必须与最低采购量匹配
- 用**老板 15% 法则**验证：单代理商合理上限 ≈ 区域年需求的 15%
- 最低采购量显著高于 15% 估算 → 🔴 或要求缩小独家区域

### 1.5 关键附件缺失

**RED 触发**（签署前必须补齐）：Schedule 产品型号与价格清单、品牌/商标使用授权范围、未填写的空白商业条款

## Tier 2 — 强烈偏好（Should-Haves）

### 2.1 最优惠价格（MFN）

- 限缩为**区域内**最优惠，禁止「全球最优惠」
- 乙方调价：提前 **30 日**书面通知；已确认订单价格不变

### 2.2 付款条件

**首选**：T/T 100% 预付（发货前）

**可接受**：部分预付 + 短账期（如 50% 预付 + 50% 见提单 30 天内）

**RED**：长账期无担保、无逾期利息、无暂停供货权

### 2.3 贸易术语（Incoterms 2020）

**首选**：FCA 华智融工厂 / FOB 指定港

### 2.4 终止与库存

- 终止 for convenience：乙方保留权利；甲方需提前 **90 日**通知（可谈 60 日）
- 终止 for cause：设 **15–30 日**补救期
- **终止后库存**：必须约定回购价公式或消化期

### 2.5 赔偿与责任限制

- 赔偿上限：不超过**争议订单/产品价款**
- 排除：间接损失、利润损失、商誉损失
- 质保：硬件 **12 个月**（非人为损坏）

---

## Tier 3 — 可让步（Nice-to-Haves）

- 自动续约通知期：偏好 90 日，可让步 60 日
- 保密义务存续：偏好 3 年，可让步 2 年
- 竞业限制：甲方合同期内 + 终止后 1–2 年不做竞品 **研发**

**谈判策略**：用 Tier 3 让步换取 Tier 2 中付款/终止/MFN 的改善；**绝不**用 Tier 3 换 Tier 1。

## 续签对比铁律

老客户续签时**必须先做新旧对比**（见 `distribution-review-workflow.md`）：

| 必比字段 | RED 信号 |
|----------|----------|
| 本土保护法 / 登记 | 旧无新有 → P0 |
| 最低采购量 | 涨幅是否与区域扩大匹配 |
| 独家区域 | 扩大但采购量未涨 |
| 付款条件 | 从预付改为账期 |
| 管辖法/仲裁 | 向对方国漂移 |

**案例索引**：7-Labs SRL（2018 旧约无 Law 173，2026 V2 新增 → 典型 🔴 模式）

## 升级路径

| 情形 | 动作 |
|------|------|
| 单项 🔴 可谈判 | 出 redline → 提交老板拍板 |
| 多项 🔴 或对方拒改 Tier 1 | 建议暂停签署 → 升级老板 |
| 制裁/主体异常 | 标注 → 建议外部律师或合规二次确认 |
| 涉及新国家法律 | 检索公开信息 + cheatsheet；不确定处标明「建议外部律师」 |
