# 股权激励 / 期权 Playbook（F 模式）

> **适用**：员工期权、限制性股票、股权激励计划、ESOP、高管激励协议  
> **法域**：中国法为主；境外子公司 ESOP 须叠加当地法 + 证券合规  
> **决策权**：HR/董事会起草 → 本 Agent 初审 → **必须升级外部律师**（证券法/公司法交叉）

## 与 C 模式区分

| 类型 | 模式 | 核心法规 |
|------|------|----------|
| 普通劳动合同 | C | 《劳动合同法》 |
| 期权/限制性股票/激励计划 | **F** | 《公司法》《证券法》+ 公司章程 |

## Tier 1 — 必须合规（🔴）

### 1.1 激励对象与适用性

- 激励对象应限于：董事、高级管理人员、核心技术人员/业务骨干
- **RED**：全员一刀切期权且无差异化考核
- **RED**：未上市主体向非员工第三方发期权（可能构成非法集资风险）

### 1.2 行权/归属条件

- 必须明确：归属时间表（vesting schedule）、业绩条件、离职处理
- 未归属部分：离职时通常失效或按约定回购
- **RED**：无归属条件、即时全额行权、无离职回购机制

### 1.3 定价与公允性

- 行权价格确定依据须可追溯（评估报告、董事会决议）
- **RED**：明显低于公允价值且无合理依据（税务与合规风险）

### 1.4 回购与退出

- 必须约定：离职/违约/公司回购价格公式
- 好/坏离职区分（good leaver / bad leaver）
- **RED**：公司无回购权、员工可自由转让第三方

### 1.5 控制权与稀释

- 须测算：完全稀释后股权比例、创始人控制权影响
- 须与现有股东协议/章程一致
- **RED**：激励计划与章程表决机制冲突

### 1.6 税务与申报

- 个人所得税递延纳税（如适用 101 号文精神）须符合条件
- **RED**：未提示税务后果、未约定代扣代缴

## Tier 2 — 强烈建议（🟡）

| 条款 | 要点 |
|------|------|
| 加速归属 | 并购/IPO 时是否加速；触发条件 |
| 竞业绑定 | 期权与竞业限制的衔接 |
| 保密 | 激励文件保密义务 |
| 争议解决 | 仲裁 vs 诉讼（激励争议常走商事仲裁） |
| 境外架构 | VIE/开曼 ESOP 与中国劳动合同的映射 |

---

## Tier 3 — 可协商（🟢）

- 行权窗口期长度
- 现金分红与期权关系
- 信息权范围

## 升级外部律师（强制标注）

以下情形**必须**标注「建议外部律师确认」：
- 上市/拟上市主体 ESOP
- 跨境多实体激励架构
- 涉及 38 号文/37 号文等外汇登记
- 期权池超过 15% 或影响控制权
