# POS 合同条款速查

> 经销（A）与采购（B）合同中涉及 POS 产品时的额外核对清单

## 硬件条款

| 审核项 | 标准立场 | 风险 |
|--------|----------|------|
| 产品型号清单 | Schedule 列明型号、版本、认证状态 | 🔴 空白 Schedule |
| 质保 | 12 个月免费硬件保修（非人为损坏） | 🔴 < 6 个月 |
| 认证证书 | 3C（中国）、CE/FCC（出口）、RoHS | 🔴 无认证附件 |
| 不良品处理 | RMA 流程、返修中心责任划分 | 🟡 缺失 |
| 产品责任 | 缺陷产品召回责任与费用 | 🟡 单方承担 |

## 软件 / SDK / API 条款

| 审核项 | 标准立场（我方乙方） | 风险 |
|--------|---------------------|------|
| 许可范围 | territory + term 限定；仅限推广销售目的 | 🔴 全球无限许可 |
| 反向工程 | 禁止 | 🔴 允许或归属对方 |
| IP 归属 | 华智融保留全部 IP | 🔴 IP 转让 |
| SDK 更新 | 更新义务与兼容性责任 | 🟡 缺失 |
| 开源组件 | 开源许可合规声明 | 🟡 缺失 |

**采购场景（我方甲方）**：定制开发成果归华智融；供方背景 IP 归供方。

## 品牌与商标

| 审核项 | 标准立场 | 风险 |
|--------|----------|------|
| 品牌使用授权 | 书面限定区域与期限 | 🔴 无授权附件 |
| 联合品牌 | 须书面审批 | 🟡 默示授权 |
| 品牌损害 | 对方不得贬损华智融品牌 | 🟡 缺失 |

## 技术支持与服务

| 审核项 | 标准立场 | 风险 |
|--------|----------|------|
| 技术支持范围 | 明确 L1/L2/L3 分工 | 🟡 无限技术支持义务 |
| 培训 | 培训次数与费用 | 🟡 缺失 |
| 备件供应 | 终止后备件供应期 | 🟡 缺失 |

## 数据与安全

| 审核项 | 标准立场 | 风险 |
|--------|----------|------|
| 交易数据归属 | 明确数据所有权 | 🔴 数据归对方 |
| 数据出境 | 符合当地数据保护法 | 🔴 无 DPA |
| 安全事件通知 | 72 小时内通知 | 🟡 缺失 |

详见 `pos-payment-compliance.md` 支付合规专节。
