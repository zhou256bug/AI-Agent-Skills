## Tier 1 — 必须合规（🔴 不合规不得签）

### 1.1 劳动合同法定必备条款

《劳动合同法》第17条 — 缺一项应补：

- [ ] 用人单位名称、住所、法定代表人
- [ ] 劳动者姓名、身份证、住址、联系方式
- [ ] 合同期限类型与起止
- [ ] 工作内容与工作地点
- [ ] 工作时间和休息休假
- [ ] 劳动报酬（含试用期工资）
- [ ] 社会保险
- [ ] 劳动保护、劳动条件和职业危害防护

**RED**：主体签章缺失、合同期限空白、岗位/地点过于笼统。

### 1.2 试用期

| 合同期限 | 试用期上限 |
|----------|------------|
| < 3 个月 / 以任务为期限 | 不得约定试用期 |
| 3 个月–1 年 | ≤ 1 个月 |
| 1–3 年 | ≤ 2 个月 |
| ≥ 3 年或无固定 | ≤ 6 个月 |

- 试用期工资 ≥ 同岗位最低档的 **80%**，且 ≥ 当地最低工资
- **同一用人单位与同一劳动者只能约定一次试用期**

### 1.3 工时与加班

- 标准工时制为默认；综合计算工时、不定时须**经劳动行政部门审批**
- **RED**：「自愿无偿加班」「放弃加班费」类条款

### 1.4 社保与公积金

- 依法参加社会保险为法定义务，不得约定「自愿放弃社保」

### 1.5 违法解除/违约金

**用人单位不得约定的违约金**（除两种法定情形）：
1. 服务期培训协议（专项培训费用 + 服务期）
2. 竞业限制（符合法定条件的劳动者 + 支付经济补偿）

**RED**：普通岗位约定离职违约金；竞业限制未约定补偿。

## Tier 2 — 强烈建议（🟡）

### 2.1 工作内容与地点

- 岗位名称 + 主要职责描述；避免「公司有权任意调岗至任何岗位」

### 2.2 薪酬结构

- 基本工资 + 绩效/津贴拆分清晰；发薪日、支付方式

### 2.3 保密

- 保密范围、期限（可长于劳动关系）

### 2.4 竞业限制

**适用对象**：高级管理人员、高级技术人员、其他负有保密义务的人员。

**必须约定**：竞业范围、期限 ≤ **2 年**、**经济补偿**、违约金与补偿的平衡

**RED**：对普通销售/文员签竞业且无补偿。

### 2.5 培训服务期

- 须证明**专项培训费用**（有凭证）
- 违约金 ≤ 培训费未履行部分

### 2.6 规章制度援引

- 劳动合同可援引员工手册，但手册须**民主程序 + 公示**

## Tier 3 — 可协商（🟢 常规）

- 通讯工具使用、知识产权归属（职务成果归用人单位）
- 离职交接流程、档案转移时限
- 争议解决：劳动争议**仲裁前置**

## 解除与终止专节

| 情形 | 核对要点 |
|------|----------|
| 劳动者辞职 | 提前30日（试用3日）书面通知 |
| 协商解除 | 双方签字，经济补偿另行约定须合法 |
| 过失性解除（39条） | 证据充分性提示 |
| 无过失解除（40条） | 提前30日通知或+1月工资；经济补偿 |
| 裁员（41条） | 🔴 升级：建议外部律师 |
| 违法解除后果 | 继续履行或赔偿金（2N） |

**离职文书三件套**：解除/终止协议、离职证明、工作交接清单。

## 劳务合同 / 顾问协议防劳动关系

核对是否构成事实劳动关系（人格/组织/经济从属性）：
- 避免：固定考勤、岗位考核、内部工牌、长期独家服务
- 🟡 建议：项目制交付、自主安排时间、可服务其他客户

**RED**：名为劳务实为劳动，未缴社保的合规风险。

## 升级外部律师

- 境外当地雇佣（非中国法域）
