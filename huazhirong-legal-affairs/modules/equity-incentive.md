# F 模式 — 股权激励 / 期权

> 触发：期权、股权激励、ESOP、限制性股票、行权、vesting、激励计划

## 加载清单

1. `references/equity-incentive-playbook.md`
2. `frameworks/legal-reasoning.md`
3. 如涉及劳动关系衔接 → 简引 `references/labor-cn-playbook.md` §竞业

## 执行步骤

### 1. 确认文本类型

| 类型 | 审核重点 |
|------|----------|
| 期权授予协议 | 行权价、归属、离职处理 |
| 限制性股票 | 解禁条件、回购 |
| 激励计划（整体） | 池比例、治理审批、税务 |
| 劳动合同中的期权条款 | 与独立协议一致性 |

### 2. Tier 评估

**Tier 1 必查**：激励对象、归属条件、定价依据、回购机制、控制权稀释、税务提示

### 3. 强制升级标注

F 模式输出**必须**包含「建议外部律师确认」段落（证券法/公司法交叉）。

### 4. 输出

按 `review-output-template.md` F 模式，归档 `output/合同/股权/审核记录/`。

## 与 C 模式边界

- 普通岗位劳动合同 → C
- 含期权/限制性股票/激励计划 → F（可同时提示 C 模式竞业衔接）
