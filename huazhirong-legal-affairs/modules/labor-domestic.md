# C 模式 — 中国境内用工

> 触发：劳动合同、试用期、竞业、社保、员工手册、境内用工  
> 我方角色：**用人单位**

## 加载清单

1. `references/labor-cn-playbook.md` — 全文
2. `frameworks/legal-reasoning.md`

## 执行步骤

### 1. 确认分类

填 `contract-routing.md` → 确认「境内用工 / 用人单位」。

### 2. 识别文本类型

| 类型 | 审核重点 |
|------|----------|
| 劳动合同 | 法定必备条款、试用期、社保 |
| 保密/竞业 | 适用对象、补偿、期限 |
| 劳务/顾问 | 防劳动关系认定 |
| 离职文书 | 程序合规、证明内容 |
| 员工手册 | 民主程序 + 公示 |

### 3. Tier 评估

**Tier 1 必查**：
- 法定必备条款完整性
- 试用期上限与工资
- 社保义务
- 违法违约金

**Tier 2 重点**：
- 竞业限制对象与补偿
- 调岗条款合理性
- 规章制度援引

### 4. 输出

按 `review-output-template.md` C 模式输出，归档 `output/合同/人事/审核记录/`。

## 升级条件

- 裁员（41条）→ 建议外部律师
- 群体性纠纷 → 建议外部律师
- 高管股权激励 → P2 延后，标注升级
