# G 模式 — 合资 / 股东协议 / 增资

> 触发：JV、合资、股东协议、增资、股权转让、章程修订、股东借款、出资方案

## 加载清单

1. `references/corporate-jv-playbook.md`
2. `frameworks/legal-reasoning.md`
3. 如涉及当地子公司 → 简引 `data/jurisdiction-index.json`

## 执行步骤

### 1. 确认交易类型

| 类型 | 审核重点 |
|------|----------|
| 增资协议 | 出资额、方式、稀释、资金用途 |
| 股东借款协议 | 利率、期限、还款来源、税务 |
| JV/合资协议 | 治理、表决权、业务划分、退出 |
| 股权转让 | 定价、优先购买权、审批 |
| 章程修订 | 与股东协议一致性、当地 mandatory rules |

### 2. 出资结构分析（常见）

对比股东借款 vs 增资的税务与治理影响，给出表格建议。

### 3. Tier 评估

**Tier 1 必查**：出资方式、治理/保留事项、退出机制、争议解决、外汇/ODI 提示

### 4. 案例参照

墨西哥子公司增资场景见 `corporate-jv-playbook.md` §案例锚点。

### 5. 输出

按 `review-output-template.md` G 模式，归档 `output/合同/股权/审核记录/`。

## 升级条件

- 跨境 ODI/外汇 → 建议外部律师
- 控制权变更 → 升级老板 + 外部律师
