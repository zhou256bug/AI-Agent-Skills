# E 模式 — 解约与终止

> 触发：解约、终止、顾问协议解除、通知期、锁定期、协商解除

## 加载清单

1. `references/termination-strategies.md` — 策略与案例
2. 视合同类型追加：
   - 经销合同 → `negotiation-playbook.md` §2.4
   - 采购合同 → `procurement-playbook.md` §1.8
   - 劳动合同 → `labor-cn-playbook.md` §解除与终止
3. `frameworks/legal-reasoning.md`

## 执行步骤

### 1. 确认合同类型与角色

先分类（经销/采购/用工/顾问），再读对应 Playbook 终止条款。

### 2. 提取关键条款

| 条款 | 摘录 | 影响 |
|------|------|------|
| 通知期 | | |
| 锁定期 | | |
| 赔偿/和解 | | |
| Survival 条款 | | |
| 身份/代表变更 | | |

### 3. 时间线分析

- 最早可发出通知的日期
- 最早可生效终止的日期
- 通知期内的费用义务
- 锁定期与通知期的交互（通知 ≠ 解除）

### 4. 方案对比

至少给出 2–3 个方案（协商解除 / 通知程序 / 争议解决），对比成本、退出日、风险。

### 5. 输出

按 `review-output-template.md` E 模式输出。

## 案例锚点：巴西顾问解约

见 `termination-strategies.md` §案例：
- 锁定期 2025.9.9–2026.9.8
- 6 月可发通知，终止日设在 9 月 10 日后
- 推荐协商解除 $55K vs 纯通知 $82.5K

## 跨文化提示

涉巴西等高 UA 文化时，可在输出中注明「建议配合跨文化沟通策略」，但**本 Agent 不委派**其他 skill。

## 升级条件

- 对方主张额外赔偿且无合同依据 → 评估诉讼风险，建议外部律师
- 涉及董事/工商变更 → 提示同步办理
