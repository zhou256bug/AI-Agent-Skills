# Kluckhohn 六维价值取向

> 文化人类学视角,补充 Hofstede 未覆盖的维度。
> 当问题涉及"做事方式 / 时间观念 / 身体语言 / 制度信任"时,优先用本框架。

| 维度 | 两极 | 商业含义 |
|------|------|---------|
| **1. 人性本质** | 性本善 ↔ 性本恶 | 性本善→法条笼统、信任人际调节;性本恶→法条事无巨细、依赖制度约束 |
| **2. 人与环境** | 主宰自然 ↔ 和谐共生 | 主宰→积极改造、目标导向管理;共生→顺势而为、关系导向 |
| **3. 人与他人** | 个人 ↔ 集体 | 与 Hofstede individualism 一致。"我是谁?——用差异定义 vs 用社会角色定义" |
| **4. 活动取向** | 行动派 ↔ 三思后行 | 行动派→快速试错、Doer culture;反思型→先分析再行动 |
| **5. 时间观念** | 着眼未来 ↔ 着眼过去 | 未来导向→创新驱动;过去导向→传统尊重、品牌历史溢价(如法国) |
| **6. 空间观念** | 公私区分高 ↔ 公私区分低 | 区分高→一臂距离、办公室文化;区分低→拉美近距离、身体接触频繁 |

**典型搭配**:
- 法国:着眼过去(品牌历史)+ 反思型 → 工匠精神溢价
- 美国:主宰自然 + 行动派 + 着眼未来 → 创新驱动 + 快速迭代
- 日本:和谐共生 + 反思型 + 公私区分高 → 严谨 + 含蓄 + 礼仪繁复
- 拉美:公私区分低 + 现在导向 → 身体接触多 + 当下享受
