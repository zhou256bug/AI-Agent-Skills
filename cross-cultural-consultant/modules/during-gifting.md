# Module:出国中-送礼场景

## 触发

- "送礼" / "带什么礼物" / "礼物"
- "伴手礼" / "见面礼" / "纪念品"
- "礼物价位" / "送什么合适"

## 输出模板

```markdown
🎁 **[国家] 送礼 - 直接给答案**

**1. 推荐(3 选)**
- 选项 A:[具体礼物品类] - 价位区间 [XX-XX 元 / 美元]
- 选项 B:[具体礼物品类] - 价位区间 ...
- 选项 C:[具体礼物品类] - 价位区间 ...

**2. ❌ 千万别送**(2-3 条具体禁忌)
- [禁忌物 1] → 原因:[宗教/文化/数字/颜色等具体理由]
- [禁忌物 2] → 原因:...

**3. 怎么送**(时机 + 包装 + 话术)
- **时机**:[初次见面送 / 离别送 / 饭局送]
- **包装**:[颜色禁忌 / 是否当面拆 / 是否双手递]
- **话术**:"[递礼时该说的一句话]"

⚠️ **特殊提醒**(若适用):
- 价位红线:[公司层面是否有反贿赂合规要求]
- 性别区分:[男性/女性客户的差异]

---
→ 还有其他场景问题吗?
```

## 关键规则

1. **价位给区间**,因为用户预算不同
2. **禁忌写具体原因**:不要只说"不要送钟",要写"钟谐音'终',预示生命终结——这条在中港台都适用,但西方国家无此忌讳"
3. **宗教国家特别检查**:
   - 伊斯兰国家:绝不送酒、猪皮制品、含猪油食品
   - 印度教国家:绝不送牛皮制品
   - 佛教国家:佛像不能随便送(尊敬度问题)
4. **颜色禁忌**:
   - 日本:白色用于葬礼,礼物包装避开
   - 中国/港台:白色/黑色包装避开,红/金为吉
   - 巴西:紫色与死亡相关
5. **数字禁忌**:
   - 中港台:避 4(死)、喜 6/8/9
   - 日本:避 4(死)、9(苦)
   - 西方:避 13
6. **合规提示**(对中国出海企业重要):
   - 美国 FCPA:对政府关系类客户礼物有严格限额
   - 德国/北欧:商业礼物通常 < 50 欧元,超过需报税

## 数据调用

读取该国 Hofstede 数据,主要看:
- **集体主义** → 是否送全办公室分享型礼物
- **PD** → 是否需要给老板和下属差异化礼物
