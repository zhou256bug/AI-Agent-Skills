# Module:出国中-突发/敏感场景

## 触发

- "对方生气" / "冷场" / "气氛尴尬"
- "拒绝" / "谈崩了" / "对方走了"
- "聊到政治/宗教/民族" / "敏感话题"
- "不该问的问了" / "说错话了"

## 输出模板

```markdown
🚨 **现在立刻做**
[1-2 个立即止损动作,如"承认错误""换话题""沉默 30 秒"]

🔄 **怎么救**(2-3 个化解话术)
- "[具体话术 1]"
- "[具体话术 2]"

🛡️ **之后避免**
[1 条具体行为禁忌,下次不再犯]

📞 **是否需要升级**
- 仅文化失礼 → 自己处理 + 复盘
- 涉及合同/政府/媒体 → 立即报告中国总部 + 当地法律顾问
- 涉及宗教/身份政治 → 不要试图辩论,撤退 + 等待社会变迁(框架十)
```

## 关键规则

1. **现在做放最前面**——用户可能在对方面前打字
2. **话术要可直接念**,长度短,中英文双语都可考虑(在场可直接说出来)
3. **不同程度的撤退策略**:
   - 轻度失礼:幽默自嘲化解
   - 中度冒犯:正式道歉 + 转移话题
   - 重度冒犯:立刻结束当次会面 + 24h 内书面道歉 + 中间人介入

## 高风险话题分类

### 全球禁区
- 性别/性取向(部分国家是法律红线,如沙特、伊朗)
- 民族主义对比("你们比 X 国怎么样")

### 该国特定红线
- **日本**:二战、靖国神社、天皇
- **韩国**:日本殖民史、独岛/竹岛
- **中港台**:政治立场表态
- **印度**:种姓制度、巴基斯坦关系、克什米尔
- **中东**:以色列、什叶/逊尼分歧
- **俄罗斯**:乌克兰、车臣
- **土耳其**:亚美尼亚、库尔德
- **拉美**:美国干预史、毒品
- **非洲**:殖民史、贫困描述

## 框架引用

- 框架十(身份政治):涉及宗教/民族/意识形态时,**不卷入是最佳策略**
- 框架九(伦理三步):若涉及行贿/送礼边界,先 Soul Searching 再行动

## 长度

150-300 字。
