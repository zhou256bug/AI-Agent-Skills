# Module:出国前(画像 + 踩坑清单)

## 触发

- "下周去 X 国" / "我要去 X 出差" / "给我 X 国画像"
- 只给国家名,无具体场景
- "去 X 要注意什么"(无附加场景细节)

## 输出模板

```markdown
# [国家名] 文化画像(出国前)

## 一、文化速览卡(中国对照)

| 维度 | 中国 | [目标国] | Delta | 商业含义 |
|------|------|----------|-------|---------|
| 权力距离 | 80 | XX | ±X[标记] | [一句话] |
| 个人/集体 | 43 | XX | ±X[标记] | ... |
| 事业成功/生活质量 | 66 | XX | ±X[标记] | ... |
| 不确定性规避 | 30 | XX | ±X[标记] | ... |
| 长期/短期导向 | 77 | XX | ±X[标记] | ... |
| 放纵/克制 | 24 | XX | ±X[标记] | ... |

> Delta 标注:> 20 = 注意 | > 30 = ⚠️ 高风险

**最危险的碰撞点**:[Delta 最大的 1-2 个维度,一句话点出现实冲突]

## 二、沟通密码

- **语境类型**:[查 frameworks/hall-context.md 参照表]
- **见面/会议风格**:[2-3 句具体动作]
- **谈判节奏**:[快推 vs 慢磨 / 关系优先 vs 合同优先]

## 三、中国人最容易踩的 3 个坑

针对该国 Delta 最大维度,给 3 条**具体可观察的行为禁忌**:

1. **[具体动作]** → 对方解读:[文化解读] → 后果:[实际反应]
2. **[具体动作]** → 对方解读:[...] → 后果:[...]
3. **[具体动作]** → 对方解读:[...] → 后果:[...]

要求:必须是动作级(送礼/握手/座次/某句话/邮件用词),不是抽象原则。

## 四、出国前 checklist

✅ **送礼**:能送什么 / 不能送什么 / 价位区间 / 包装与颜色
✅ **餐桌**:菜系偏好 / 酒类规则 / 谁先举杯 / 禁忌话题
✅ **见面**:握手力度 / 名片递接 / 座次 / 称呼
✅ **时间观**:对方迟到几分钟正常 / 你能迟到几分钟 / 会议节奏
✅ **节日/季节提示**(若用户给了具体时间):该季节宗教节日、气候、避忌

## 五、教授框架点睛

从 frameworks/12-frameworks.md 中选 1-2 条最相关:
- [框架名]:[一句话判断 + 行动指令]

---

> 数据快照:见 data/hofstede-dimensions.json `scraped_at` | 数据反映国家层面统计倾向,个体差异始终存在
> 出国中遇到具体场景(餐桌/谈判/送礼/突发),直接问我,可连续追问。
```

## 长度

1200-1800 字。Delta >30 的维度必须出现在"碰撞点"和"踩坑清单"。
