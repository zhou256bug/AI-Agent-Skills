# Module:出国中-会议/谈判中实时解读

## 触发

- "对方刚说..." / "客户这么说"
- "对方沉默" / "不点头" / "笑而不语"
- "对方突然问..."
- "怎么接话"

## 输出模板

```markdown
🎯 **[国家] 实时解读**

**对方真实意思**
[文化解读 - 一句话点穿对方的潜台词]

**你应该回**
"[具体话术]"

**节奏建议**
[追问 / 等待 / 换话题 / 立即记笔记]

⚠️ 不要做:[1 条该情境下中国人本能反应但会踩雷的事]
```

## 关键规则

1. **极快响应**——这个模块用户是现场打开手机,3 秒内要看到答案
2. **真实意思要"点穿"**,不要绕弯:
   - 日本"我考虑一下" → "这是委婉拒绝,但不要立刻撤退,先问'如果有 X 调整是否可行'"
   - 印度点头 → "可能是听到了,不是同意,追问'那下一步具体怎么走'"
3. **节奏建议要给动作**,不是"保持冷静"这种没用的话
4. **不要做**写具体动作:
   - "不要笑场"
   - "不要急着填空"
   - "不要在群里追问"

## 高语境国家专属规则

日/中/韩/越/阿拉伯/希腊:
- 沉默 = 思考,等 10-15 秒再说话
- 笑 = 礼貌,不代表同意
- "我们再研究"= 99% 拒绝
- 点头 ≠ 同意,只是"我听到了"

## 低语境国家专属规则

美/德/北欧/荷兰:
- 沉默 = 等你说话,不是拒绝
- 直接反驳 ≠ 不尊重,反而是认真对待
- "Maybe" = 真的 maybe,不是委婉 no
- 不要用"我们关系不错吧"建立信任,他们要看合同和业绩

## 长度

100-250 字。一句话能解决就一句话。
