# B 模式示例：墨西哥、埃及、孟加拉三国对比

**场景**：菜头刚从孟加拉回来，要求对比三国看他实际感受是否对得上数据
**模式**：多国对比模式

## 核心发现

1. **权力距离三国几乎一致**（80/80/81）——这是三国唯一没有明显差异的维度
2. **最大差异在放纵指数**：墨西哥 97 → 埃及 0 → 孟加拉 20
   - 墨西哥：全球最放纵，家庭时间不可侵犯，工作是为了生活
   - 埃及：全球极少数的 0 分克制社会，严肃务实
   - 孟加拉：高度克制，公共场合拘谨
3. **集体主义梯度**：孟加拉 5（极端集体）→ 埃及 13 → 墨西哥 34
   - 孟加拉的 Aamra 客户决策受家族圈子影响极大
4. **不确定性规避差异**：墨西哥 82 → 孟加拉 60 → 埃及 55
   - 墨西哥需要写死合同，孟加拉可商量，埃及最灵活

## 关键教学点

- 在三国间切换时，最需要注意的不是权力距离（都一样），而是"放纵指数"的悬崖式跳跃
- 墨西哥的"先喝酒再聊生意"套路，在埃及和孟加拉完全不适用
- 孟加拉的家族圈做生意模式 vs 墨西哥的个人意识更强
