# B 模式示例：孟加拉 vs 印尼 文化对比 + 业务策略

**场景**：中国管理者评估孟加拉手机分期项目，要求对比印尼的差异性
**模式**：多国对比模式，含中国基线

## 核心发现

1. **集体主义两国高度一致（均为5）**——全世界最低档，个人不存在，一切以家族/社群为单位
2. **最大差异在放纵指数（Delta 18 ⚠️）**：孟加拉 20（极度克制）vs 印尼 38（较放纵）
   - 这是对分期/消费金融影响最大的维度
   - 印尼消费者天然愿意为当下的满足付费，分期推广阻力小
   - 孟加拉对"分期"本身持怀疑态度——为什么要为今天的消费付利息？
3. **不确定性规避差异（Delta 12）**：孟加拉 60 vs 印尼 48
   - 孟加拉做新业务需要权威背书（央行/大银行），合同必须极其详细
   - 印尼灵活，有示范案例后可快速跟进
4. **长期导向两国都比中国低得多（中国77 vs 孟38 vs 印尼29）**——印尼是三者中最短视的

## 业务策略对照表（手机分期项目适用）

| 环节 | 孟加拉 | 印尼 |
|------|--------|------|
| 合作获客 | 走银行/政府背书，层层审批 | 走代理商网络，先小规模试点 |
| 分期产品设计 | 6-12个月，低利率，伊斯兰金融优先 | 3-6个月，可接受稍高费用 |
| 地推团队 | 本地大代理+宗教社区领袖 | 直接招本地团队，B2B2C更有效 |
| 风控重点 | 身份证+家庭住址+担保人 | 社交网络+POS交易历史 |
| 宗教考量 | 伊斯兰金融是必要条件 | 伊斯兰金融重要但不绝对 |
| 时间感 | 慢——2-3次见面才有信任 | 中——1-2次见面可推进 |

## 两国共同的坑（中国管理者高发）

- **别用"中国速度"要求**：孟加拉慢是因为高UA（层层确认怕犯错），印尼慢是因为低LTO+高放纵（不着急）
- **集体主义不是团结——是圈子化**：跟一个人谈好了换部门又从头来；关系是对"那个人"的，不是对"那个公司"的
- **面子文化**：两国面子可能比中国还强（个人主义5分），公开丢脸＝整个家族丢脸
- **点头不一定同意**：跟印度一样，上下晃头是常见体态

## 一句话总结

> 孟加拉＝**保守的集体主义者**：需要权威背书+严谨合同+长期耐心才能推进。
> 印尼＝**灵活的集体主义者**：关系到位后推进速度更快，但持续性可能不如孟加拉。
