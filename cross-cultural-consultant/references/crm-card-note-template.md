# 客户名片笔记模板（C 模式会后 · 单 Agent 写入）

本技能内 CRM 产物统一落在 `output/crm/`（已在 `.gitignore`）。**由当前 Agent 直接写文件**，不委派外部 Agent 或依赖其他 skill。

## 文件路径约定

| 类型 | 路径 |
|------|------|
| 名片原图 | `output/crm/附件/{姓名}_{公司}.jpg` |
| 名片笔记 | `output/crm/客户名片/{大洲}/{国家}/{国家}-{公司}-{姓名}.md` |
| 客户主档案 | `output/crm/客户信息/{国家}-{客户}.md` |
| 会议纪要 | `output/crm/Meetings/{国家}-{客户}-meeting-{YYYY-MM-DD}.md` |

`{大洲}` 示例：`南美洲`、`亚洲`、`欧洲`。

## 更新客户主档案

在 `output/crm/客户信息/{国家}-{客户}.md` 的「联系人」表追加一行，并链到名片笔记文件名。若主档案不存在，Agent 应先创建带表头的文件再追加。
