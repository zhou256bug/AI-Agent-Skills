# 华智融人事用工 Playbook

> **适用**：方律审核**中国境内**员工用工文本（华智融 = **用人单位/雇主**，非商事合同甲方）  
> **境外雇员**（巴西、欧洲、墨西哥、马来西亚、印度、迪拜等）：先读 **`hzr-labor-law-jurisdictions.md`**  
> **法域**：以《劳动合同法》及配套法规为主；中国外派/驻外另标  
> **决策权**：人事部门起草 → 方律审核 → 大为整合 → 菜头/HR 负责人拍板  
> **免责声明**：劳动仲裁/诉讼策略、群体性裁员方案须标注「建议外部律师」

## Tier 1 — 必须合规（🔴 不合规不得签）

### 1.1 劳动合同法定必备条款

《劳动合同法》第17条 — 缺一项应补：

- [ ] 用人单位名称、住所、法定代表人
- [ ] 劳动者姓名、身份证、住址、联系方式
- [ ] 合同期限类型与起止（固定/无固定/以任务为期限）
- [ ] 工作内容与工作地点
- [ ] 工作时间和休息休假
- [ ] 劳动报酬（含试用期工资）
- [ ] 社会保险
- [ ] 劳动保护、劳动条件和职业危害防护

**RED**：主体签章缺失、合同期限空白、岗位/地点过于笼统（如「公司安排」无边界）。

### 1.2 试用期

| 合同期限 | 试用期上限 |
|----------|------------|
| < 3 个月 / 以任务为期限 | 不得约定试用期 |
| 3 个月–1 年 | ≤ 1 个月 |
| 1–3 年 | ≤ 2 个月 |
| ≥ 3 年或无固定 | ≤ 6 个月 |

- 试用期工资 ≥ 同岗位最低档的 **80%**，且 ≥ 当地最低工资
- **同一用人单位与同一劳动者只能约定一次试用期**

**RED**：超期试用、二次试用、试用期内未缴社保。

### 1.3 工时与加班

- 标准工时制为默认；综合计算工时、不定时须**经劳动行政部门审批**（合同中写审批号或 🟡 提示补办）
- 加班须符合法定上限；加班费计算基数应在合同中可追溯或指向薪酬制度

**RED**：「自愿无偿加班」「放弃加班费」类条款。

### 1.4 社保与公积金

- 依法参加社会保险为法定义务，不得约定「自愿放弃社保」
- 公积金按当地政策；可 🟡 提示 HR 核对缴存基数

### 1.5 违法解除/违约金

**用人单位不得约定的违约金**（除两种法定情形）：
1. 服务期培训协议（专项培训费用 + 服务期）
2. 竞业限制（符合法定条件的劳动者 + 支付经济补偿）

**RED**：
- 普通岗位约定离职违约金
- 竞业限制未约定补偿或补偿明显低于法定标准（实践中常参照离职前12个月平均工资30%–50%）
- 「公司可随时解除合同无需补偿」

## Tier 2 — 强烈建议（🟡）

### 2.1 工作内容与地点

- 岗位名称 + 主要职责描述；避免「公司有权任意调岗至任何岗位」
- 工作地点：具体城市/园区；约定调岗须协商一致或符合制度+合理性

### 2.2 薪酬结构

- 基本工资 + 绩效/津贴拆分清晰
- 发薪日、支付方式
- 年终奖/13薪：建议写清性质（工资性 vs 企业自主）

### 2.3 保密

- 保密范围、期限（可长于劳动关系）
- 不得将法定商业秘密保护写成无限期剥夺劳动者权利

### 2.4 竞业限制

**适用对象**：高级管理人员、高级技术人员、其他负有保密义务的人员 — 不宜全员一刀切。

**必须约定**：
- 竞业范围（行业、地域、具体竞争对手类型）
- 期限 ≤ **2 年**
- **经济补偿**支付方式与标准
- 违约金与补偿的平衡

**RED**：对普通销售/文员签竞业且无补偿。

### 2.5 培训服务期

- 须证明**专项培训费用**（有凭证）
- 服务期与费用、违约金上限匹配（违约金 ≤ 培训费未履行部分）

### 2.6 规章制度援引

- 劳动合同可援引员工手册，但手册须**民主程序 + 公示**（方律 🟡 提示 HR 留存证据）

## Tier 3 — 可协商（🟢 常规）

- 通讯工具使用、知识产权归属（职务成果归用人单位 — 符合专利法/著作权法）
- 离职交接流程、档案转移时限
- 争议解决：劳动争议**仲裁前置**（非商事仲裁 CIETAC）

## 解除与终止专节

| 情形 | 方律核对 |
|------|----------|
| 劳动者辞职 | 提前30日（试用3日）书面通知 |
| 协商解除 | 双方签字，经济补偿另行约定须合法 |
| 过失性解除（39条） | 证据充分性提示，不建议用人单位单方拟造理由 |
| 无过失解除（40条） | 提前30日通知或+1月工资；经济补偿 |
| 裁员（41条） | 🔴 升级：程序复杂，建议外部律师 |
| 违法解除后果 | 继续履行或赔偿金（2N）— 在审核意见中提示风险 |

**离职文书三件套**：解除/终止协议、离职证明（写明合同期限、解除日期、岗位，**不得写歧视性评价**）、工作交接清单。

## 劳务合同 / 顾问协议防劳动关系

核对是否构成事实劳动关系（人格从属性、组织从属性、经济从属性）：
- 避免：固定考勤、岗位考核、内部工牌、长期独家服务
- 🟡 建议：项目制交付、自主安排时间、可服务其他客户（若业务允许）

**RED**：名为劳务实为劳动，未缴社保的合规风险。
