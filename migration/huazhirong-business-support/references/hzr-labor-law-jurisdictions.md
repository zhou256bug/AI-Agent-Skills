## §巴西

**法律框架**：Consolidação das Leis do Trabalho (CLT)；集体谈判协议（CCT）可能优于法定最低。

| 审核项 | 要点 | 风险 |
|--------|------|------|
| 合同形式 | 书面；关键条款葡语 | 🟡 仅英文无葡语 |
| 工资与福利 | 13º salário、FGTS（8%）、休假 + 1/3 津贴 | 🔴 遗漏法定福利 |
| 工时 | 44h/周；加班须补偿 | 🔴 无偿加班条款 |
| 解雇 | 正当 cause vs 无正当理由； severance / multa 40% FGTS 等 | 🔴 我方单方无补偿解除 |
| 试用期 | 90 天（可续一次，共 90） | 超期试用 |
| 数据 | LGPD 雇员数据 | 无隐私告知 |

**RED 触发**：合同排除 CLT 强制权利、PJ（个体户）安排掩盖劳动关系（pejotização）。

**升级**：集体解雇、工会谈判、劳动法院诉讼 → 巴西当地律师。

## §欧洲

**法律框架**：欧盟层面指令（工作时间、平等、跨国信息咨询）+ **成员国国内法**（德国 Kündigungsschutz、法国 CDI、荷兰等差异极大）。

### 欧盟共性初审

| 审核项 | 要点 |
|--------|------|
| 书面条款 | 雇佣后短期内须提供核心书面条件（欧盟 91/533/EEC 精神） |
| 工作时间 | 休息、年假（法定最低 4 周） |
| GDPR | 雇员数据处理合法性、跨境传输 |
| 解雇 | 多数成员国解雇保护强 — **不得**写「随意 at-will 解雇」 |
| 竞业 | 各国可执行性差异大；须补偿且期限合理 |

### 国别提示（签约前确认具体国家）

| 国家 | 特别注意 |
|------|----------|
| 德国 | 解雇社会正当性；固定期合同续期限制；Works council |
| 法国 | CDI 默认；试用期严格；集体协议（CCN）优先 |
| 荷兰 | 链式固定合同 → 自动永久；30% ruling（税务，转 HR） |

**RED**：适用美国 at-will 模板、无年假、无解雇程序。

**升级**：裁员、Works council、跨境派遣（Posted Workers）→ 该国律师。

## §墨西哥

**法律框架**：Ley Federal del Trabajo (LFT)；2021 外包改革（禁止将核心员工外包为 subcontracting）。

| 审核项 | 要点 | 风险 |
|--------|------|------|
| 书面合同 | 西语；必备条款（工资、日程、地点等） | 缺失法定内容 |
| 利润分享 | PTU（员工分红）义务 | 🔴 合同排除 PTU |
| 解雇 | 正当 cause 或 indemnización（约 3 个月工资 + 20 天/年等，依情形） | 🔴 零补偿解除 |
| 工时 | 48h/周；加班费率 | 违规放弃加班 |
| 外包 | 核心岗位不得通过第三方 employer of record 规避 | 🔴 虚假外包 |

**升级**：大规模解雇、工会、STPS 检查应对 → 墨西哥律师。

## §马来西亚

**法律框架**：Employment Act 1955（2022 修订扩大适用）；外籍员工另需工作准证。

| 审核项 | 要点 | 风险 |
|--------|------|------|
| 适用性 | EA 适用门槛（月薪等）— 高管可能不适用部分条款 | 误用模板 |
| 终止通知 | 合同/法定通知期；有因 vs 无因 | 🔴 即时解雇无程序 |
| 加班 | 适用 EA 者工时上限 | 违规 |
| 外籍员工 | Employment Pass、配额、终止与离境 | 无许可工作 |
| 公积金 | EPF/SOCSO（适用时） | 未缴 |

**升级**：Industrial Court 争议、集体协议 → 当地律师。

---

## §印度

**法律框架**：联邦 + 邦法； Shops & Establishments、Industrial Disputes Act、Contract Labour Act；IT/制造业常遇双重结构。

| 审核项 | 要点 | 风险 |
|--------|------|------|
| 任命函 / 合同 | 岗位、薪酬、地点、保密 | 与 shop act 登记一致 |
| 终止 | 多数需 notice 或代通知金；有因解雇须程序与证据 | 🔴 at-will |
| 固定期合同 | 适用条件严格；续期限制 | 非法连续固定期 |
| 公积金 | PF、ESI（门槛适用时） | 未缴 |
| 竞业 | 印度法下竞业限制可执行性有限 — 依赖保密而非宽泛竞业 | 过宽竞业 |
| 合同工 | Contract Labour (Regulation and Abolition) Act 许可 | 非法派遣 |

**升级**：劳动专员调查、工业争议、邦法差异 → 印度律师。

## §迪拜 / 阿联酋

**法律框架**：Federal Decree-Law No. 33 of 2021（UAE Labour Law）；MOHRE 注册；**大陆法区 vs 自由区（Free Zone）** 规则可能不同。

| 审核项 | 要点 | 风险 |
|--------|------|------|
| 合同登记 | MOHRE 标准合同 / 登记要求 | 未登记风险 |
| 工资 | WPS（工资保护系统）支付 | 现金/off-WPS |
| 工时与加班 | 法定上限；斋月等 | 违规放弃 |
| 终止 | 正当 cause vs arbitrary dismissal；notice（30–90 天依类型） | 🔴 零通知零补偿 |
| 终服金 | EOSB（gratuity）按连续服务计算 — 不得 waiver | 🔴 放弃 gratuity |
| 竞业 | 须限定时间/地点/活动 + 可能需补偿 | 过宽无效 |
| 签证绑定 | 离职与签证取消衔接 | 劳动纠纷与移民交叉 |

**自贸区**：核对具体 FZ 规章（如 DIFC 有独立雇佣法）— **签约前确认 Free Zone 名称**。

**升级**：MOHRE 投诉、arbitrary dismissal 赔偿诉讼 → 阿联酋律师。

## 跨境用工模式核对

| 模式 | 方律核对 |
|------|----------|
| 华智融中国主体雇中国员工 | `hzr-labor-contract-playbook.md` |
| 华智融境外子公司当地雇当地员工 | 本文件对应国 + 当地律师 |
| EOR/PEO（名义雇主） | 🔴 审查 EOR 协议 + 实际用工国法律；核心岗位外包风险 |
| 中国外派至境外 | 派遣合规、社保、个税、当地工作许可 |
| 独立承包商 | 各国「假自雇」测试 — 见各国 §RED |
