## PDF 生成工作流

1. 写 HTML（390px宽，NotoSansSC字体嵌入，中文内容+菜头英文开场白）
2. `weasyprint input.html /tmp/temp.pdf`（不走pdfprint——它有完整HTML二次包装导致字体覆盖的bug）
3. `gs ... -sOutputFile=output.pdf /tmp/temp.pdf`（GS优化）
4. 用 PyMuPDF 裁剪底部空白：
   ```python
   import fitz
   doc = fitz.open('output.pdf')
   page = doc[0]
   blocks = page.get_text('blocks')
   min_y = min(b[1] for b in blocks if b[6] == 0)
   max_y = max(b[3] for b in blocks if b[6] == 0)
   crop_rect = fitz.Rect(0, max(0,min_y-5), page.rect.width, min(page.rect.height, max_y+5))
   page.set_cropbox(crop_rect)
   doc.save('cropped.pdf')
   ```
   **不要设固定@page height（会导致分页或底部空白）**，用超大高度（2000mm）保证一页，然后用PyMuPDF精确裁剪。
5. 存 `newpos/出差报告/` + 微信发菜头（注意微信iLink限速，多次发送失败时用cron job延迟重试 `in 30m`）
