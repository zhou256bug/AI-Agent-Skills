# 中国企业工商注册状态查询流程

## 触发条件

需要验证某家公司在中国大陆（特别是深圳）是否正式注册：

⚠️ 当菜头问"查一下XX公司在深圳的注册状态"，这属于企业工商注册状态查询，**不是一般背景调研**。查询优先级高于写背景笔记，结论应简洁直接（已注册/未查到），附带排除的近似公司名。

## 实际查询流程（逐步扩展）

### 第一层：搜索引擎初筛

直接搜索完整公司名，用引号：
```
"深圳声智互联科技有限公司" 注册
"深圳声智互联科技有限公司" 企查查
```

同时尝试变体搜索以排除名称记忆偏差：
```
"声智互联" 科技 有限公司
声智互联科技 深圳
```

### 第二层：官方政府信用平台（首选）

**深圳信用网（szcredit.org.cn）** — 深圳市公共信用中心官方平台，反爬较弱，可直接浏览器交互：

```
1. 访问 https://www.szcredit.org.cn/#/index
2. 在页面上方搜索框输入完整公司名搜索
3. 如果精确搜索返回"暂无信息，共 0 条"，再用核心关键词模糊搜索
   （取公司名核心词，如"客脉"而非"客脉智能科技"）
4. 模糊搜索会列出所有含该关键词的注册企业及其状态（存续/吊销/注销）
```

**特点**：
- 无需登录即可查看基本信息（统一社会信用代码、状态、主体类型、成立日期）
- 模糊搜索可发现近似公司名 — 帮助排除"记错了名字"的可能性
- 搜索结果直观区分"存续/吊销/注销"标签
- 登录后可查看更多信息（法定代表人、注册地址等）

### 第三层：商业平台交叉验证

分别搜索以下平台，作为官方验证后的交叉确认：
1. 企查查 (qcc.com) — 更新最快（T+1），但反爬较严
2. 天眼查 (tianyancha.com) 
3. 爱企查 (aiqicha.baidu.com) 

**反爬策略**：各平台均有反爬机制（验证码、HTTP 405、操作异常检测）。验证码/空页/405 本身也是信号——如果公司存在，首次搜索通常会直接进入公司详情页而非反爬页。主要依赖搜索引擎初筛和官方平台确认，商业平台作为补充。

### 第四层：近似公司排除

搜索出同名/近似公司时，进行比较确认：
- 确认是否同一公司（如"声智科技"≠"声智互联科技"）
- 比较注册地址（北京vs深圳）、法定代表人、经营范围
- 汇总排除理由

### 第五层：确认性补充搜索

对可能接近的名称做"极限搜索"：
- 搜索"公司名 注册地址"、"公司名 信用代码"等
- 如无任何结果，可给出"当前未查到该名称在深圳的注册记录"的结论

## 结论判定标准

| 搜索结果 | 判定 |
|----------|------|
| 所有渠道均无结果 | **未注册/该名称可用** |
| 深圳信用网有记录 + 模糊搜索发现近似公司但无完全匹配 | **该精确名称未注册，但有近似公司存在，需确认是否记错名** |
| 企查查/天眼查有工商信息页 | **已注册，可查看详细状态** |
| 只有近似名，无完全匹配 | **该精确名称未注册** |
| 有搜索结果但均为无关内容 | **未注册或刚注册未被收录** |
| 曾注册但显示"注销"/"吊销" | **已注销/吊销，名称不再可用** |

## 结果呈现

结论先行，陈述：已查 / 未查 / 核实不误。

**已查到时呈现**：
- 统一社会信用代码
- 法定代表人
- 成立日期
- 注册资本
- 注册地址
- 经营状态（存续/在营/注销/吊销）

**未查到时的表述**示例：
> 所有渠道（深圳信用网 + 搜索引擎 + 三大商业征信平台）均无记录。
> 模糊搜索发现近似公司：[列出近似公司名+状态] ，均不是目标公司。
> 该名称在当前可用。

## Pitfalls

- **深圳信用网优先于商业平台**：这是官方平台，反爬弱，可直接交互产生结果
- **先精确搜索后模糊搜索**：精确搜索"暂无信息"后立即用核心词模糊搜索，这是验证名称是否记错的关键步骤
- **反爬不等于无记录**：政府网站（gsxt.gov.cn）和商业征信平台都有反爬，但大量渠道均无结果本身就是明确信号。不需要绕过反爬去查——官方平台+多个搜索引擎+三个商业平台都没结果就是最明确的信号
- **gsxt.gov.cn 的特性**：全国企业信用信息公示系统需要 JS 渲染，curl 返回 521 是正常现象（非防火墙拦截），需浏览器访问。对于纯命令行环境，它的爬取价值低，优先用深圳信用网代替
- **容易混淆相似名**：区分不同城市的同名公司
- **分词搜索陷阱**：搜索引擎可能分词匹配出片段，这些不是匹配结果，要检查完整公司名
- **跨区域同名干扰排除**：搜索"聆动科技"等热门行业词时，可能匹配到外地的同名/近似公司（如安徽聆动通用机器人科技、合肥聆动通用等）。需逐一排除并告知用户"有外地同名公司，但深圳无注册记录"
- **精确匹配才有效**：公司名含"智能"与否是不同实体（如"深圳聆动科技有限公司"≠"深圳市聆动智能科技有限公司"）。前缀"深圳"vs"深圳市"在工商系统中是独立区分的名称。用户提供的写法必须严格按其说明，不要自动补充"市"或"智能"
- **刚注册的新公司**：注册后1-2周内可能还未被所有平台收录，企查查/天眼查通常更新最快（T+1）
- **正式注册前的最终确认**：必须通过深圳市市场监督管理局的"企业名称自主申报"系统做最终校验，上述渠道仅供参考
