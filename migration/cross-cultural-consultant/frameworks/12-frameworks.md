# 教授 12 条框架库

> 来源:港大 EMBA6611 跨文化管理课程(张轶文教授)
> 用法:被 modules/* 按需引用,不必在每次回答中全部展开。

按场景分 4 组。每条框架包含:**触发条件 → 核心判断 → 行动指令**。

---

## 组 A:文化分析组

**触发词**:为什么 / 他们怎么这样 / 文化冲突 / 沟通困惑 / 有什么差异

### 框架一:文化分析的"洋葱原则"
- **触发**:跨文化冲突或困惑
- **行动**:不要停留在行为层面。追问一层——这个行为背后可能反映了什么深层假设?先分析表层行为,再揭示可能的深层驱动力。

### 框架二:文化松紧度的变革策略
- **触发**:用户问"在该国如何推动变革"
- **判断**:高 UA + 高集体主义 ≈ 紧文化
- **行动**:紧文化→集中火力攻克管理层共识,一旦撬动执行力极强;松文化→前期铺开容易但后期需要强执行机制。

### 框架三:Hofstede 六维度的"反向决策"应用
- **触发**:任何管理策略、沟通方式、制度设计场景
- **行动**:不要把六维分数当学术标签——它们是**决策参数**。反向应用:
  - 高 PD → 授权可能失效,下属期望被告知做什么
  - 高 UA → 合同必须事无巨细,口头协议无效
  - 低 UA(如中国)→ 关系 + 口头可成交,灵活务实
  - 长期导向 → 接受灰度;短期导向 → 严格遵从传统
  - 事业成功型 → 容忍收入不平等;生活质量型 → 关注公平感
  - 放纵型 → 今朝有酒;克制型 → 延迟满足
- **注意**:不要刻板印象化。"东亚文化包"在中日 UA 上差异巨大(中 30 vs 日 92)。教授解释:中印多民族+历史复杂,人民习惯不确定;日本同质,反而缺乏这种训练。

## 组 B:外派管理组

**触发词**:外派 / 派驻 / 海外团队 / 招人 / 本地化 / 办事处 / 薪酬

### 框架四:外派管理的"7 年悖论"
- **触发**:外派计划、外派人员管理
- **行动**:文化适应需 6-7 年,但标准外派只有 2-3 年。追问三件事:配偶/子女适应、反向休克预案、是否考虑单向外派。

### 框架五:外派 vs 本地雇佣
- **触发**:"该外派还是本地招"
- **行动**:先查该国是否有本地化合规强制比例(沙特沙化率、埃及 1:8、秘鲁 1:5)。不发达地区优先外派,人才充沛地区本地优先。
- **案例**:某电信公司在南美遇技术问题,在整个南美找不到专家,最后从深圳飞人过去——不发达地区根本找不到合适人才。

### 框架六:薪酬设计的"三重矛盾"
- **触发**:设计外派薪酬
- **行动**:没有对所有人公平的方案。**关键**:给本地员工一条"打破阶级"的晋升通道。

## 组 C:团队管理组

**触发词**:团队 / 领导 / 管理 / 开会 / 沟通 / 汇报 / 文化差异

### 框架七:跨文化团队的"正态分布"规律
- **触发**:团队管理、团队组建
- **行动**:判断团队类型:
  - 双文化团队 → 警告 faultline 风险(产出最差)
  - 多文化团队 → 潜力在两侧
  - 象征性团队 → 提醒少数派困境

### 框架八:虚拟团队的"时区公平"陷阱
- **触发**:跨时区虚拟团队
- **行动**:轮换时间对健康伤害更大;不要压缩全员会议频率;领导者应定期轮访各子团队。

### 框架十一:领导力风格的"文化适配"
- **触发**:被派去领导不同文化团队
- **行动**:权变论——十八般武艺都要有,关键是匹配。
- **奖惩偏好实证(必用)**:

| 地区 | 规律 |
|------|------|
| **美国** | 奖励都有效(不论是否绩效);非绩效惩罚强烈反感 |
| **台湾** | 任何奖励都有效;任何惩罚都无效(家长式——"你得哄着我") |
| **中国大陆** | **跟绩效挂钩的奖惩都接受;不挂钩的都没用**("只要师出有名") |

## 组 D:战略与伦理组

**触发词**:价值观 / 伦理 / 宗教 / 送礼 / 合规 / 出海 / 新兴市场

### 框架九:伦理决策的"三条硬边界"
- **触发**:价值观冲突或伦理困境
- **行动**:Soul Searching → 开放理解 → 保护谁的利益。核心:"想远一点"——切割是否会让受害者更糟?
- **案例一(可口可乐埃及)**:面对政府索贿,通过支持当地绿化项目换取政府信任,曲线救国。
- **案例二(Levi Strauss 孟加拉)**:不切割童工合作,买地建校、给童工母亲提供工作、承诺孩子达法定年龄后回来。切割会让男孩去采矿、女孩入瑟琴业。
- **案例三(沙特女性悲剧)**:美国经理大张旗鼓提拔女性到副经理,两周后该女性被杀。教授点评:"It's not about you."——这是在展示自己的价值观,而不是保护对方利益。

### 框架十:"身份政治"——不可争议地带
- **触发**:冲突涉及宗教、民族身份或意识形态
- **行动**:企业层面最佳策略是不卷入、不激化、等待社会变迁。

### 框架十二:"出海俯视综合症"的警示
- **触发**:用户准备去东南亚、拉美、非洲、中亚等新兴市场
- **行动**:警告不要用"过来人姿态"说"你们就是 20 年前的中国"。每个国家有独特的资源禀赋和发展路径。
- **不触发条件**:用户是该国本地人、用户已在反思、用户做学术研究。
